# Resonance Lab

명조: 워더링 웨이브의 보유 캐릭터, 육성 상태와 선호도를 기록하고 여러 파티를 구성하는 Python 웹 앱입니다.

## 실행

외부 패키지가 필요하지 않습니다.

```bash
python3 server.py
```

브라우저에서 `http://127.0.0.1:8000`을 엽니다.

## 현재 기능

- 2026-06 기준 캐릭터 53명과 초상 이미지
- 이름·속성·무기 검색 및 필터
- 보유 여부, 공명 체인, 레벨, 육성 상태, 최대 사용 횟수, 전용 무기와 재련 저장
- 레벨 `MAX 90` 단축 버튼과 사용 횟수 기본값 1
- SQLite 로컬 영구 저장
- 메타 파티·피해 유형·버프 태그·육성도·무기를 함께 반영한 복수 파티 전체 최적화
- 모바일 반응형 UI

## 구조

- `server.py`: 정적 파일 서버, JSON API, SQLite 저장, 추천 엔진
- `data/characters.json`: UI와 추천 엔진이 공유하는 캐릭터 카탈로그
- `data/team_rules.json`: 검증된 메타 파티와 피해/버프 호환성 지식 베이스
- `static/`: 프레임워크 없는 웹 UI

캐릭터 이미지는 nanoka.cc 의 공개 이미지 경로를 참조합니다. 배포/상용화 전에는 쿠로게임즈 팬 콘텐츠 정책과 각 데이터 제공처의 이용 조건을 별도로 확인하세요.
